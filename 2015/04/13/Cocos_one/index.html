Cocos_js游戏开发

　　一个月前，简单的学习了一下配置以及第一个程序。然后一个月后竟然忘记了，这里花时间来复习一下。

介绍：

　　Cocos是一个游戏开发的就架。可以使用C，C++，JS来开发。然后配置好一些相应的工具包，就可以实现，转换为在任何平台使用的程序。一次编译，永久使用。

　　这里有一个很好很全的教程：

　　Cocos_js开发教程

　　然后关于里面的几点，我大体说一下。

  • 里面的webstorm 是一个js的开发软件，和Cocos Code IDE似乎重复，其实是有那么一个感觉。你可以选择性使用。

  • 关于里面的snk,sdk,python等等的配置，都是为了以后编译，模拟，转换做准备。

  • 关于使用命令行的位置：就是在你的工程目录的下面调出cmd就行了，

  • 你要是使用js开发游戏只需要下载：
    Cocos2d-JS 和 Cocos Code IDE就行了。前者解压后是一个文件（js框架），后者在配置框架时会会需要他的解压路径。

  • 所以前者的解压路径，不要出现中文或者括号等等。
